# Churn Propensity Model

End-to-end churn prediction system

## Stack
- BigQuery (feature store, partitioned + clustered)
- Colab (EDA + training notebooks)  
- XGBoost + MLflow (modelling + experiment tracking)
- Airflow via Docker (pipeline DAG)
- FastAPI + Cloud Run (model serving)
- GitHub Actions (CI/CD)
- Evidently (monitoring)

## Business context
Telco customer churn prediction across 7,043 customers.
